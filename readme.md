# Arquitetura de Software – Partes Difíceis (Prática)

Este repositório é dedicado à prática dos conceitos apresentados no livro  
**“Software Architecture: The Hard Parts”**.

A proposta é transformar a teoria em experimentação, por meio de pequenos exemplos
e implementações que exploram decisões arquiteturais, trade-offs e problemas reais
enfrentados no design de sistemas.

Projeto com finalidade exclusivamente educacional.
